# SandUhr 2.0

PlotClock-Derivat mit Griffel und Sandbett

3D-Design erstellt mit kostenlosem 3D-CAD DesignSpark Mechanical 2.0, Gerber-
Files für kleine Servo-Interface-Platine mit RTC, kann auch auf Lochraster 
aufgebaut werden.

Für 3D-PDF wird neuerer Acrobat Reader benötigt. Zeichnungen für Lasercutter als PDF und CorelDraw-Datei (min. Version 12).

Basiert auf PlotClock vom FabLab Nürnberg, http://wiki.fablab-nuernberg.de/w/Ding:Plotclock
